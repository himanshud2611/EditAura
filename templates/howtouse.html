How to Use EditAura

  1. Select an Image
    Click on the "Choose File" button and select the image you want to edit from your device.
  2. Choose an Operation
    From the dropdown menu, select one of the following operations:
    • Convert to PNG
    • Convert to Grayscale
    • Convert to Webp
    • Convert to Jpg
  3. Submit
    Click the "Submit" button to process your image.
  4. Download Your Edited Image
    After processing, you'll see a success message with a link to your edited image. Click the link to view and download your processed image.

Supported File Types

EditAura supports the following image file types:

  • PNG
  • WEBP
  • JPG/JPEG
  • GIF

Tips

  • Make sure your image is in one of the supported file formats.
  • For best results, use high-quality original images.
  • The processed image will be available for a limited time, so make sure to download it promptly.
